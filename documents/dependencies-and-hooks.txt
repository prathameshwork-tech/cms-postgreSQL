# DEPENDENCIES AND HOOKS DOCUMENTATION

## FRONTEND DEPENDENCIES

### Core Dependencies
- **React**: JavaScript library for building user interfaces
- **Vite**: Build tool and development server
- **@mui/material**: Material UI React components
- **@mui/icons-material**: Material UI icon set
- **@emotion/react** & **@emotion/styled**: CSS-in-JS for MUI
- **axios**: HTTP client for API calls
- **react-router-dom**: Client-side routing
- **react-hook-form**: Form handling and validation
- **recharts**: Charting library for React
- **react-toastify**: Toast notifications
- **classnames**: Utility for conditionally joining classNames
- **papaparse**: CSV parsing
- **@headlessui/react**, **@heroicons/react**: UI primitives/icons (if used)

### Development Dependencies
- **@vitejs/plugin-react**: Vite plugin for React
- **eslint** and plugins: Code linting
- **autoprefixer**, **postcss**: CSS tooling
- **tailwindcss**: (Remove if not in use)
- **@types/react**, **@types/react-dom**: TypeScript types (if using TS)
